Criar um site com três páginas:

Página 1 - Explicar o que é uma publicação de páginas estáticas

Uma publicação de páginas estáticas se refere à criação de sites web cujo conteúdo não muda dinamicamente. Essas páginas são armazenadas no servidor web e servidas aos usuários sem necessidade de processamento adicional no servidor.

Página 2 - Explicar o que é o Ubuntu e a principal vantagem de usá-lo como servidor de sites dinâmicos

O Ubuntu é um sistema operacional de código aberto, baseado no Linux, que é amplamente utilizado como servidor web. Uma das principais vantagens de usar o Ubuntu como servidor de sites dinâmicos é a sua estabilidade, segurança e facilidade de configuração e manutenção.

Página 3 - Explicar alguns comandos básicos de ubuntu

Alguns comandos básicos do Ubuntu incluem:

  • sudo - Permite executar comandos com privilégios de administrador
  • apt-get - Gerenciador de pacotes para instalar, atualizar e remover software
  • ls - Lista os arquivos e diretórios em um diretório
  • cd - Navega entre os diretórios
  • mkdir - Cria um novo diretório
